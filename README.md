# TS_Bitwise_Example
:page_with_curl: Example of how to use bitwise flags with typescript

A simple example of how to use bitwise flags with typescript.

To run with node:
`tsc .\index.ts; node .\index.js`

To run with deno:
`deno .\index.ts`
